## 2. 회차 요약

- **제목**: 최약체의 귀환 · EP01 (시즌1 1막 오프닝 / 무료 후킹의 정점 · 시리즈 방어선)
- **로그라인**: 세계 최약체로 버림받아 죽었다가 각성 직전으로 회귀한 헌터 서도현이 자신을 짓밟은 가해자들을 냉정하게 참교육하려 하지만, 회귀로 얻은 미래 기억은 힘을 쓸 때마다 한 조각씩 지워져 — 복수를 끝내기 전에 '왜 돌아왔는지'조차 잊을 위기에 직면한다.
- **반전 한 줄**: 각성은 "최약체 고정"이라는 세계 규칙의 **예외**였고(T1 사이다), 그 힘의 대가는 **미래 기억을 태워 없애는 것**이며(T2 숨은 비용), 그의 미래에 유일하게 빈칸인 존재 '예언'이 그의 회귀를 이미 알고 있다(T3 클리프행어).
- **반전 3연타**: T1 규칙 반전(각성=정보로 산 예외) → T2 숨은 비용(미래 소모·자기부식 F-04) → T3 정체 예고 클리프행어(예언 실루엣 + 도현이 잊은 '가려진 조각' 재봉인).
- **패널 수**: **66장** (`panel_001`~`panel_065` + P24 분할 `024a`/`024b`, 결번 없음).
- **긴장 곡선**: 절망낙차(1막) → 각성/사이다(2막 T1) → 소모·예언 접근·클리프행어(3막 T2·T3) 우상향. P65 풀블랙 암전으로 3중 훅 미회수 상태 컷.

## 4. 다음 회차(EP02) 시드 — 클리프행어 이어받기 포인트

EP01의 클리프행어와 미회수 떡밥을 EP02 시작점으로 넘긴다. 정본(series-arc §4 EP02 행, twist-plan §5)과 정합. **continuity.md 확정 시 continuity-manager와 최종 합의해 원장에 기록**하고, 시나리오팀(series-plotter·twist-master)에 인계한다.

### 이어받기 훅 (EP01 종료 상태 → EP02 시작)
1. **YEEON(예언) 등장 여파** — EP01 B4(P57·P61·P63)에서 역광 실루엣 + "…또 왔군." / "이번엔, 무엇을 태울 텐가."로 선지급됨(F-05·F-06 심기). **정체 미확정 유지가 규약**(world §5). EP02에서 예언을 직접 재등장시키지 말 것 — 정식 재등장은 EP26(실루엣)·EP30(직접). EP02는 예언 대사가 도현 내면에 남긴 균열("나를 아는 자가 있다")의 **여파**만 다룬다.
2. **T3 미회수 훅 — 잊힌 '가려진 조각'(F-01/F-02)** — 도현이 T2로 태워 사라진 미래 조각(P64 "…방금 내가, 뭘 하려고 했지.")이 B0의 "그것만은 잊으면 안 돼"였을 가능성을 재봉인한 채 컷. → EP02 핵심 사건 "첫 죽음 회상에서 '빈 구멍' 발견 → 자기 기억 불신"으로 직결. **최상위 빚(MW-1/MW-2)은 시즌3까지 미회수 유지.**
3. **F-04 소모 제약 이월** — EP01은 BURN(태움) 체감까지 부분회수. EP02는 DRIFT(아는 미래가 어긋남) **예고 씨앗**을 심어 EP05 DRIFT 확립으로 연결(series-arc §5).

### EP02 시작점 (series-arc §4 확정 골격)
- **핵심 사건**: 미래 노트 정리·최적 각성 계획 수립(작은 통제 회복) + 첫 죽음 회상에서 '빈 구멍' 발견(위협: 자기 기억 불신).
- **등장**: 도현·곽노인.
- **심는 떡밥**: F-03(첫 죽음 진상 균열)·F-11(빈 구멍의 가짜 기억, 신뢰불가 화자 씨앗).
- **반전 슬롯**: 과거 진상 균열.
- **EP02 클리프행어**: 아는 미래와 어긋난 사소한 사건 하나 발견(RULE-DRIFT 예고).
