## 2. 회차 요약

- **제목**: 최약체의 귀환 · EP03 (시즌1 1막 / EP02 클리프행어 직결 · 최적 게이트 잠입 준비 + 한소이 재회·거래 + F-07 2차 노출 · 최봉규 팀 동일 게이트 배정 확인)
- **로그라인 정합**: 힘을 쓸 때마다 미래 기억이 지워지는 서도현이, 게이트 잠입을 위해 유일하게 자신을 사람으로 대하는 힐러 한소이와 거래로 팀을 이룬다 — 그러나 그녀의 손등 힐링 문양은 EP01보다 더 흐려져 있고("다 꺼지면 넌 죽어"), 배정 명단엔 죽여야 할 표적 최봉규가 같은 게이트로 올라 있다.
- **회차 성격**: 현재 시점 실등장 3인 = **도현·한소이·곽노인.** 최봉규는 P54(명단 이름 베이크만·얼굴 미등장)+P56(원경 실루엣만)의 **간접 처리**(직접 등장 아님, 혼동 분리 성공). 전투 없음. 힘 사용은 "운용"(P5 미래 지식·P46 루트)으로만 연출(재각성 없음), 소모 청구서(F-04)는 P6 FX_BURN 1회로 부착.
- **반전 + 클리프행어**:
  - 🔵 **F-07 2차 노출(P29~35)**: 무대사 시각 진입(P29 시선→P30 클로즈인→P31 문양 발동)으로 반전에 들어가, 도현 속말 4단(P32 "…저 문양"→P33 "능력이… 꺼져가고 있어"→P34 "그때보다… 더"→P35 "…넌 죽어")으로 의미 전달. **밝기 단조 감소가 화면으로 성립** — P31 문양은 꺼져가는 옅은 청록, P34 병치 인서트에서 기억(EP01 밝은 청록) vs 현재(흐린 청록) 대조가 한 프레임에 명확 → EP01 밝기로 되돌아오지 않음. 한소이는 자신의 약화·죽음 미래를 **여전히 자각 못 함**(정보 비대칭).
  - 🔴 **클리프행어(최봉규 동일 게이트 배정, P50~57)**: P50 예고→P51~53 명단 대조·정지→**P54 디제틱 사인 "게이트 배정 — 공격팀장 최봉규"(사실 확정)**→P55 "…최봉규. 같은 게이트라고"(자각 대사로 못 박음)→P56 원경 실루엣(혼동 분리)→P57 "…예정에, 없던 일이야"(FX_VOID+풀블랙 착지). 베이크 텍스트+자각 대사 **이중 전달**로 오독 여지 없음. **강태섭 등 상위 보스명 베이크 0건**(유보 준수).

EP03의 클리프행어와 미회수 떡밥을 EP04 시작점으로 넘긴다. 근거는 `continuity.md` "다음 회차(EP04) 주의사항" 17개 항목 + 정본 `series-arc §4·§5 EP04 행`. **continuity-manager와 합의해 원장에 기록 완료**했으며, 시나리오팀(series-plotter·twist-master)에 인계한다.

### 🔺 최우선 — series-arc §4·§5 EP04 행 정합 수정 요청 (제작 착수 전 반드시 해소)

- **문제**: series-arc §4·§5 EP04 행은 "게이트 내 최봉규가 팀을 방패로 도주하려는 순간 **미래를 태워 선각성** → 예외 등급 발현"이라 돼 있으나, **도현은 EP01에서 선각성을 이미 완료**했고 EP02에서 "각성 계획"을 "각성 후 운용"으로 재해석 확정했다(재각성 연출 금지 원칙 확립). EP04에서 "선각성"을 다시 연출하면 EP01·EP02와 모순이다.
- **요청(정본 정정)**: series-plotter·twist-master는 EP04 행을 **"이미 각성한 상태에서 최봉규의 도주 시도에 대응해 예외 등급 위력을 전면적으로 드러내는 첫 실전 시연(첫 공개 각성 예외 시연)"**으로 재해석·수정할 것. 재각성/선각성 연출 금지, 각성 완료 전제 유지. EP04의 힘 사용은 EP03과 동일하게 "운용"으로 연출한다.
- **처리 라우팅**: 이 항목은 정본 정정 사안이므로 **continuity-manager가 series-plotter·twist-master에 SendMessage로 통지**해 EP04 행을 정정하도록 요청한다(showrunner 임의 판단 금지 — 정본 정정은 시나리오팀 결정). 정정 완료 전까지 EP04 비주얼 착수 보류 권고.

### 이어받기 훅 (EP03 종료 상태 → EP04 시작)

1. **최봉규 팀 동일 게이트 배정 → 물리적 충돌 현실화** — EP03은 P57("…예정에, 없던 일이야", FX_VOID+풀블랙 암전)로 "예정 밖 변수" 불안을 열어두고 컷했다. EP04는 이 불안을 게이트 내 충돌로 현실화한다. series-arc §4상 EP04 핵심 = **"게이트 내 최봉규 도주 시도 대응 → 예외 등급 첫 전면 시연 + 은폐 물증(F-09) 확보"**, 클리프행어 = **"각성(예외 위력) 목격한 최봉규가 '속임수'라며 제거 지시 → 위기 컷오프"**.
2. **최봉규 첫 직접 대면 = 도현 정체 인지의 분기(단, 회귀는 여전히 미간파)** — EP04에서 최봉규가 도현의 예외 위력을 목격하나(EP01 P47 "E급이 어떻게?!"의 재현·심화), **도현이 회귀자임은 여전히 몰라야 함**. 제거 지시는 "속임수/이상한 놈" 수준으로, 회귀 간파 아님(회귀 정체 발각은 장기 유보).
3. **F-04 DRIFT 잔향 이월(봉합 금지)** — EP03까지 이어온 DRIFT 잔향("예정에 없던")은 밑에 유지하되, "아는 미래가 실제로 틀린다"의 본격 확립은 **EP05 유지**(EP04에서 앞당기지 말 것). EP04는 정보 우위가 아직 통하되 예외 등급 시연에 **큰 미래 소모(F-04 회수·소모 재확인)**가 붙는 단계.

### EP04 시작점 (series-arc §4 확정 골격 — 🔺 위 정합 수정 반영 전제)

- **핵심 사건**: 게이트 내부 진입 → 최봉규 도주 시도 대응 → 이미 각성한 예외 등급 위력의 **첫 전면 실전 시연** + 최봉규 은폐 물증(보고 지연 기록, F-09) 확보.
- **등장**: 도현·한소이·최봉규(첫 직접 대면·충돌). 예언·강태섭은 미등장·봉인 유지.
- **심는 떡밥**: **F-09(최봉규 은폐 물증 확보)** — 회수는 EP09 폭로(단주기, 시즌1 내 반드시 닫아 사이다 리듬 유지). EP04는 물증 "확보"까지.
- **회수 떡밥**: **F-04(소모 재확인)** — 예외 등급 시연의 큰 소모로 F-04 상시 진행을 다시 부착. DRIFT 확립(EP05)과 구분(EP04는 BURN 소모 재확인).
- **EP04 클리프행어**: 각성 예외를 목격한 최봉규가 "속임수"라며 제거 지시 → 위기 컷오프.
