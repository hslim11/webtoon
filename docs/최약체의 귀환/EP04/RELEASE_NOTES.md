## 4. 다음 회차(EP05) 시드 — 클리프행어 이어받기 포인트

EP04의 클리프행어와 미회수 떡밥을 EP05 시작점으로 넘긴다. 근거는 `continuity.md` "다음 회차(EP05) 주의사항" 18개 항목(★DRIFT 확립 최우선 구획 포함) + 정본 `series-arc §4·§5 EP05 행`. **continuity-manager와 합의해 원장에 기록 완료**했으며, 시나리오팀(series-plotter·twist-master)에 인계한다.

### ★ 최우선 — DRIFT 확립 빌드업 회수 (EP02~EP04 3회차 인계 터뜨리기 · 시즌1 최초 중간점 반전)

- **EP05는 시즌1 최초의 중간점(midpoint) 반전 회차다.** EP02(예고·"첫 금")→EP03(잔향)→EP04(잔향 심화·P57 "…이 다음은, 안 봤어")로 **3회차에 걸쳐 밑에 깔아온 DRIFT를 마침내 터뜨려야 한다.** 지금까지 "본격 확립은 EP05로 유보"라고 계속 미뤄 온 것을 **여기서 정식 확립** — **"태운 미래의 하류 정보가 처음으로 실제와 안 맞음"(RULE-DRIFT 첫 목격)**을 화면·대사로 확정. **이 빌드업이 또 미뤄지면 3회차 인계가 방치 떡밥이 된다.** series-plotter·twist-master가 반드시 확인·집행할 것(회수 실패 = 시리즈 최상위 리스크).
- **"이미 봤어"가 통하지 않는 순간을 명확히 연출** — EP01~EP04 도현의 정보 우위 시그니처("봤어/이미 봤어/안다")가 EP05에서 **처음으로 빗나가** 도현이 역으로 궁지에 몰린다(정보 우위 부식 자각). **EP04 P44 "네가 어디서 무너질지, 난 이미 봤어"의 자신감과 대비되는 붕괴**로 설계 — 사이다(EP04) 직후의 통제감을 EP05에서 무너뜨리는 반전 리듬.
- **통제감 붕괴 = 중간점 반전의 심리 축** — EP01~EP04에서 쌓아온 "작은 통제 회복"이 EP05에서 무너진다. 막1의 중간점 2차 피크(tension-engineer 인계)이므로 긴장 곡선상 국소 고점.
- **DRIFT ≠ VOID 시각 구분 계승·확립**: DRIFT는 **반투명 잔상**(있어야 할 정보가 부재/다름, EP02 P55 자전거 유령 문법), VOID는 **먹칠 빈 구멍**(기억 소실). EP05 DRIFT 정식 확립 시 "반투명 잔상=DRIFT" 문법을 계승해 시각 구분.

### 이어받기 훅 (EP04 종료 상태 → EP05 시작)

1. **게이트 내 위기 직결 → 함정 역이용으로 컷오프 해소** — EP04는 **최봉규 "속임수" 제거 지시(P58 "죽여") → 팀원 쇄도(P59) → 도현 "…예정보다, 빠르군"(P60, 블랙컷·이중 착지)**로 위기 컷했다. EP05 시작은 그 위기 직결이며, series-arc §4상 도현이 **미래로 최봉규 함정을 역이용(작은 우위)해 일단 넘긴다.** 시간선은 EP04 게이트 내부에서 연속(회귀 기준 2016-04-15에서 계산).
2. **함정 역이용이 통하다가 → 태운 미래의 하류 정보가 처음 안 맞음(낙차)** — "운용이 통하다가 처음 실패하는" 낙차를 설계. EP04의 "예정보다 빠른 충돌"과 "이 다음은 안 봤어"를 도입에서 함정 역이용으로 일단 넘기되, **그 다음 정보가 빗나가는** 지점에서 DRIFT를 확립한다. 클리프행어 = **"믿은 미래가 빗나가 도현이 역으로 궁지 — '이미 봤어'가 통하지 않음"**.
3. **최봉규 정체 인지 분기 유지(회귀는 여전히 미간파)** — EP05에서 함정 역이용·충돌이 격화돼도 **최봉규는 도현을 회귀자로 간파하지 않는다**("속임수/변수" 프레임 유지, 회귀 발각은 장기 유보). 도현이 회귀자임 발설(T3)도 금지.
4. **F-04 질적 전환** — EP05는 F-04 **회수(DRIFT 확립)** 회차. 소모가 "부식(정보가 틀림)"으로 **질적 전환**됨을 화면으로. EP01 심기→EP05 확립→이후 회차마다 누적 증폭. "대가 없는 승리" 금지.

### EP05 시작점 (series-arc §4·§5 확정 골격)

- **핵심 사건**: (게이트 내 위기 직결) → 미래로 최봉규 **함정 역이용(작은 우위)** → 그러나 태운 미래의 하류 정보가 처음 안 맞음 → **RULE-DRIFT 확립·정보 우위 부식 자각·"이미 봤어" 불통** → 통제감 붕괴.
