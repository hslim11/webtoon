EP02(예고·"첫 금")→EP03(잔향)→EP04(잔향 심화·"이 다음은, 안 봤어")로 **3회차에 걸쳐 밑에 깔아온 RULE-DRIFT를 마침내 정식 확립**한다 — "태운 미래의 하류 정보가 처음으로 실제와 안 맞음"을 화면·대사로 확정하고, EP01~EP04 도현의 정보 우위 시그니처("이미 봤어")가 **처음으로 빗나가** 도현이 역으로 궁지에 몰린다. 이 회차가 미뤄지면 3회차 인계가 방치 떡밥이 되는 시리즈 최상위 리스크였으며, EP05가 이를 정확히 집행해 **F-04를 부분회수→확립 완료로 최종 전환**했다. 시리즈 골격상 놓쳐선 안 될 회차이므로 품질·연속성 게이트를 특히 엄격히 본 뒤 승인한다.

  - **(a) 도현 왼목 흉터 재발(EP02 P27·P38 사고 유형)**: EP05는 심리붕괴+액션 다패널 고위험 회차였으나 `matte-black high-neck suit, NO neck scar` 앵커가 전 도현 패널에서 작동해 하이넥 칼라가 목을 완전히 덮어 death-version 흉터 **미노출 전수 클린**(P47·P62 ECU에서도 노출 0). EP02→EP03→EP04→EP05 **4연속 재발 0건**으로 앵커 규칙 유효성 계속 실증.
  - **(b) 도현↔최봉규 근접 공존 혼동**: 공존 5컷(P10·22·43·56·58)이 4대 대비축(체형 마름 vs 우람 183 / 피부 창백 쿨톤 vs 그을린 붉은기 / 표정 무표정·붕괴 vs 비웃음·광기 / 식별 표식 도현 없음 vs 최봉규 오른눈썹 흉터·왼목 늑대 타투·견장·도그택)으로 **전 컷 명확 분리**. 뒤섞임·오혼입 0, 최봉규 회귀 간파 표정 0(EP04에 이어 2연속 분리 성공).
  - **(c) FX_DRIFT(신규 이펙트) ≠ FX_VOID 혼동**: DRIFT는 **반투명 이중노출 phantom/파편**(P27·31·33·37·46·52 아크), VOID는 **불투명 차콜 먹칠+모자이크**(P1·60)로 같은 화면에서 섞이지 않고 명확히 구분된다. EP02 P55 자전거 유령 문법을 계승해 신규 FX_DRIFT 토큰이 캐논으로 정착.
  - **(d) 재각성 오독**: 함정 역이용은 "이미 각성한 힘의 운용"으로 연출되고 **재각성 이펙트 0**. P37·40·41 도현 눈의 옅은 바이올렛 광채는 DRIFT 예지-시야 부식 악센트(눈 국한·잔상 동색)이지 신체 발광 오라/각성 폭발이 아님(validation C4·QA §6 이중 확인).
- **반전 이중 전달 화면 성립**: 💠 RULE-DRIFT 확립(반투명 잔상 아크 P27→33→37→46→52)과 🔴 "이미 봤어" 3단 붕괴(P30 자신감 정점→P34 첫 균열→P45 불통 확정)가 대사·시각·사건(P38 죽은 개체 생존·P41 코어 재활성) 3축 동시 하강으로 성립. 클리프행어 P62("…안 봤는데.")가 EP04 P57("이 다음은, 안 봤어")의 대구로 착지.
- **회차 간 연속성(continuity.md)**: EP05 갱신 블록이 이어써졌고 **미해결 모순 없음**. EP01~EP04 원장(회귀 상태 흉터 없음·매트블랙 전투복 디폴트·각성 완료 전제·재각성 금지·T3 발설 금기·FX 문법·DRIFT≠VOID 구분·강태섭 유보)과 EP05 산출물이 전부 정합. **F-04가 부분회수→확립 완료로 최종 전환**(3회차 빌드업 회수·이후 상시 진행), 새로 심긴 빚 없음(F-07 강화 심기는 EP06 예정). **미회수 방치 떡밥 없음.**

## 2. 회차 요약

- **제목**: 최약체의 귀환 · EP05 (시즌1 1막 / **중간점 반전** · EP04 게이트 내부 제거 지시·쇄도 클리프행어 직결 → 미래로 최봉규 함정 역이용(작은 우위)·소형 사이다 → 그러나 태운 미래의 하류 정보가 처음 '안 맞음' → **RULE-DRIFT 정식 확립·정보 우위 부식 자각·"이미 봤어" 불통** → 통제감 붕괴·역궁지)
- **로그라인 정합**: 힘을 쓸 때마다 미래 기억이 지워지는 서도현이, 게이트 내 위기를 미래 지식으로 최봉규를 함정에 몰아 일단 넘기지만("…작게, 태웠어.") — 이번엔 **태워서 어긋난 미래의 하류 정보가 처음으로 실제와 안 맞고**("왜… 안 맞아. 분명, 봤는데."), 죽었어야 할 몬스터 코어가 재활성하며 도현이 역으로 협공에 몰린다. 최봉규는 여전히 회귀를 간파하지 못한 채 "그 '봤다'는 소리, 이제 안 나오네?"로 기세를 뒤집는다.
- **회차 성격**: 게이트 내부 단일 씬(SC01 · LOC_GATE_INTERIOR variant D) 심리붕괴+액션 회차. 현재 시점 실등장 = **도현·한소이(도입부 3컷·대피만)·최봉규**. 힘 사용은 "이미 각성한 힘의 운용"으로 연출(재각성/선각성 이펙트 0 — 각성 완료 전제 계승). 소모 청구서(F-04)는 이 회차에서 **"대가(BURN·기억이 탐)"에서 "위협(DRIFT·태운 미래가 틀림)"으로 질적 전환**됨을 P47 "…내가 태워서, 어긋난 건가."로 도현이 처음 자각.
- **반전 + 클리프행어**:
  - 💠 **RULE-DRIFT 확립(반투명 잔상 문법)**: 화면으로 성립 — P27(무릎 꿇는 최봉규 원본 미래를 반투명 이중노출 phantom으로 도입·복선) → P33 full-bleed(잔상 무릎 vs 서서 저항하는 실제 어긋남·첫 확정·"빌어? 내가 왜!") → P37(반투명 유리질 파편으로 부서짐) → P46(대량 반투명 파편·부식 절정) → P52(형체 못 이루는 반투명 안개 부재). 전량 translucent 문법으로 렌더, 먹칠 아님. 독자는 "있어야 할 미래가 부재/다름"을 화면으로 읽는다.
  - 🔴 **"이미 봤어" 3단 붕괴**: P30(자신감 정점·안정 정면·firm "…봤어. 넌 여기서 무너져") → P34(첫 균열·dutch·속말 "…어?") → P45(불통 확정·"왜… 안 맞아. 분명, 봤는데.")로 자신감→균열→붕괴가 대사·시각·사건 3축 동시 하강. EP04 P44 "네가 어디서 무너질지, 난 이미 봤어"의 자신감과 정면 대비되는 통제감 붕괴.
  - 🔴 **클리프행어(역궁지)**: P56~58 최봉규 위계 역전("지금은 내가 위야."·"'봤다'…이제 안 나오네?") + 재활성 몬스터↔최봉규 협공(P60 FX_VOID 먹칠·강습) + P61 "이 다음을… 난—"(말 끊김) → **P62 full-bleed 도현 눈 ECU→풀블랙 암전·"…안 봤는데."**(EP04 P57 대구로 착지). 강태섭 등 상위 보스명 베이크 0건(유보 준수).
- **패널 수**: **62장** (`panel_001`~`panel_062`, 결번 없음).
- **긴장 곡선**: 콜드오픈(P1 EP04 P60 블랙컷 직결·리셋 없음·긴장 8) → 함정 발동 소형 사이다(P16~24, 7→8) → 통제 정점(P30) → 반전 확립·부식(P33~55, 8→9.5 · 중간점 반전 국소 고점) → **역궁지 클리프행어 최고조**(P56~62, 10, 하강 없이 착지) 우상향. index.html gap 리듬(하드 브레이크 0 / full-bleed 3회 체류 P33·41·62 앞뒤 64/64/200 격리 / sub-state 경계 P40→P41 64 / FX_BURN·FX_DRIFT 절정 48 격리 / 붕괴 구간 dutch 불안정 변주 16·36 / 클리프행어 조임 P56~61 전부 12 / 회차 끝 암전 여운 200)이 곡선을 지지.

## 4. 다음 회차(EP06) 시드 — 클리프행어 이어받기 포인트

EP05의 역궁지 클리프행어와 미회수 떡밥을 EP06 시작점으로 넘긴다. 근거는 `continuity.md` "다음 회차(EP06) 주의사항" 18개 항목(🔴 F-07 강화 심기 = EP06 최대 외형 리스크 포함) + 정본 `series-arc §4·§5 EP06 행`. **continuity-manager와 합의해 원장에 기록 완료**했으며, 시나리오팀(series-plotter·twist-master)에 인계한다.

### ★ 최우선 — DRIFT 확립 이후 첫 회차 운용(무쌍 아님 각인)

- **EP05는 시즌1 1막의 중간점 반전으로 RULE-DRIFT를 정식 확립했다(DRIFT 확립 완료).** 따라서 **EP06부터 DRIFT는 상시 진행·누적 증폭 규칙**이며, EP06은 "확립 대기"가 아니라 **확립된 DRIFT 상황 속에서** 진행된다. 정보 우위가 완전히 죽은 게 아니라 "부식·배신할 수 있는 것"이므로, EP06은 series-arc §4대로 **"실력 + 남은(아직 안 틀린) 정보로 간신히 역전"**을 그린다 — **먼치킨 무쌍 아님을 각인**하는 회차.
- **EP05에서 무너뜨린 통제감을 EP06에서 값싸게 회복시키지 말 것.** "이미 봤어" 한 방 사이다로 되돌아가면 중간점 반전이 무의미해진다. EP06 역전은 정보가 다시 완벽히 통해서가 아니라, 도현이 **부식되지 않은 남은 정보 + 실력(운용 판단)**으로 아슬하게 뒤집는 것이며, 역전 뒤에도 DRIFT 부식은 밑에 유지된다.
- **스케일 = 소형 역전**(series-arc §4). EP04 예외 시연급 전면 사이다가 아니라 궁지에서 간신히 빠져나오는 소형 역전. 긴장 곡선상 중간점 반전(EP05) 직후의 소폭 회복.

### 이어받기 훅 (EP05 종료 상태 → EP06 시작)

1. **역궁지 위기 직결 → 간신히 역전으로 컷오프 해소** — EP05는 **최봉규 위계 역전("'봤다'…이제 안 나오네?") + 재활성 몬스터↔최봉규 협공 + 도현 "…안 봤는데."(P62, 블랙컷·통제 붕괴)**로 위기 컷했다. EP06 시작은 그 위기 직결이며, series-arc §4상 도현이 **DRIFT 궁지에서 실력+남은 정보로 간신히 역전**한다. 시간선은 EP05 게이트 내부에서 연속(회귀 기준 2016-04-15에서 계산). 게이트 내부면 LOC_GATE_INTERIOR variant D 계열 재사용 검토.
2. **🔴 한소이 본격 재등장이 인질 정황의 무대를 연다** — EP05는 한소이가 도입부 3컷·대피만 등장했으나, EP06은 한소이 본격 재등장 회차다. 클리프행어 = **"최봉규가 한소이를 방패/인질로 삼으려는 정황"**. 이는 F-09(최봉규 은폐 물증) 대치 긴장을 높이는 소재이나, 물증을 "쓰는" 게 아니라 "쥔 채 대치가 격화"되는 선까지만(폭로는 EP09).
3. **최봉규 정체 인지 분기 유지(회귀는 여전히 미간파)** — EP06에서 협공·인질 정황으로 대치가 격화돼도 **최봉규는 도현을 회귀자로 간파하지 않는다**("속임수/변수" 프레임, 회귀 발각 장기 유보). 도현이 회귀자임 발설(T3)도 금지.
4. **F-04 상시 진행 유지·증폭** — EP05에서 확립 완료됐으므로 EP06은 성급히 "DRIFT 극복/해결"로 봉합하지 말 것(회차마다 누적 증폭이 원칙). EP06 "간신히 역전"은 DRIFT를 극복한 게 아니라 남은 정보로 우회한 것. "간신히 역전"에도 소모/부식 청구서(F-04)가 붙는다.

### EP06 시작점 (series-arc §4·§5 확정 골격)

- **핵심 사건**: (게이트 내 역궁지 직결) → **DRIFT 상황서 실력+남은 정보로 간신히 역전(무쌍 아님 각인)** → 소형 역전 → 최봉규가 한소이를 방패/인질로 삼으려는 정황으로 재긴장.
- **등장**: 도현·최봉규·**한소이**(본격 재등장).
- **심기 떡밥**: **F-07 강화**(series-arc §4 EP06 행·§5 트래커) — 한소이 문양의 **추가 흐려짐/꺼져감**으로 죽음 미래를 강화 암시. **자각은 도현 내심에만**(정보 비대칭 유지), 한소이 본인 자각·죽음 미래 발동은 EP14~15(3차 발동). EP06은 강화 심기까지, 발동 금지.
- **유보 떡밥**: F-09(폭로 EP09·무대 설계 EP08 — EP06은 물증 쥔 채 대치 격화까지) · 예언·강태섭(봉인 유지, 강태섭 첫 흘림 EP09) · F-03·F-11·장주기(F-01·F-02·F-05·F-06) 성급 회수·방치 금지. **F-08은 EP07 심기 예정**(EP06 아님).
- **EP06 클리프행어**: 최봉규가 한소이를 방패/인질로 삼으려는 정황.
