## 2. 회차 요약

- **제목**: 최약체의 귀환 · EP02 (시즌1 1막 / EP01 클리프행어 직결 · 통제 회복 시도 vs 자기 기억 불신의 심리 회차)
- **로그라인 정합**: 회귀로 얻은 미래 기억이 힘을 쓸 때마다 지워지는 서도현이, 이번엔 전투 없이 **자기 기억 자체가 신뢰할 수 없음**을 발견한다 — 첫 죽음의 진상엔 '빈 구멍'이 있고, 아는 미래는 사소하게 어긋나기 시작한다.
- **회차 성격**: 현재 시점 실등장은 **도현·곽노인 2인**뿐. 전투 없음·새 힘 사용 없음. 소모(F-04)는 "전투 대가"가 아니라 **누적 부식·금기 반동**의 형태(P3 잔재·P10 억지 상기 통증·P41 T2 위반 반동)로 부착됨.
- **반전 2종 + 예고**:
  - 🔴 **F-03(첫 죽음 진상 균열, P31~45)**: 회상 속 최봉규의 "…예, 그 지시대로"(P34 통신 보고)로 우연 방치→고의 지시 가능성. 지시 주체(강태섭)는 P36 노이즈 공백으로 완전 유보.
  - 🔴 **F-11(빈 구멍의 가짜 기억, P38~45)**: 회상 심부의 빈 구멍을 억지로 응시(T2 첫 접촉 씨앗) → 가짜 기억이 채워짐(P42 '다른 죽음' 환각). 어느 죽음도 진짜로 확정하지 않는 유보 유지.
  - 🟣 **DRIFT 예고 클리프행어(P51~57)**: 아는 미래(북구 뒷골목 사건)가 어긋남을 발견(P55 반투명 유령 자전거 잔상) → P57 "미래가 틀린 건지… 내가 틀린 건지. …모르겠어" 이중 공포·풀블랙 암전. EP05 DRIFT 정식 확립의 "첫 금".
- **패널 수**: **57장** (`panel_001`~`panel_057`, 결번 없음).
- **긴장 곡선**: 콜드오픈(P1~12, 7→5 상실·통제 상실 자각) → 상승(P13~30, 5→6 정보 거래·노트 재정비) → 반전(P31~45, 7→8 회상 균열·빈 구멍) → 클리프행어(P46~57, 6→10 통제 회복 착각→DRIFT 붕괴) 우상향. P50 "이건, 안 바뀌어"(착각 절정)를 미스디렉션 최고조로 끌어올린 직후 P53~57에서 무너뜨려 낙차 확보. P57 풀블랙 암전으로 컷.

## 4. 다음 회차(EP03) 시드 — 클리프행어 이어받기 포인트

EP02의 클리프행어와 미회수 떡밥을 EP03 시작점으로 넘긴다. 근거는 `continuity.md` "다음 회차(EP03) 주의사항" 16개 항목 + 정본 `series-arc §4 EP03 행`·`§5 떡밥 트래커`. **continuity-manager와 합의해 원장에 기록 완료**했으며, 시나리오팀(series-plotter·twist-master)에 인계한다.

### 이어받기 훅 (EP02 종료 상태 → EP03 시작)
1. **DRIFT 예고 + 자기 기억 불신 이월** — EP02는 P57("미래가 틀린 건지… 내가 틀린 건지. …모르겠어", 풀블랙 암전)로 컷했다. **이 불신을 EP03에서 완전히 봉합하지 말 것** — DRIFT 본격 확립은 EP05이므로 EP03까지 미해결로 끌고 간다. 한소이 재회로 곡선을 새로 열되 자기 불신의 잔향을 밑에 깐다.
2. **F-03·F-11(EP02 정식 심기 완료) 유보 유지** — EP03에서 성급히 회수하지 말 것. 어느 죽음이 진짜인지·지시 주체(강태섭)가 누구인지 답을 주지 않는다(회수: F-03 EP26~27→시즌3, F-11 시즌3). 첫 죽음 관련 컷에서 강태섭 이름·형체는 노이즈 공백 유지.
3. **최봉규 팀 동일 게이트 배정(EP03 클리프행어 예고)** — series-arc §4상 EP03은 이 충돌의 "예고"까지만. EP04 게이트 내 선각성·충돌로 이어진다.

### EP03 시작점 (series-arc §4 확정 골격)
- **핵심 사건**: 최적 게이트 잠입 준비 + 한소이와 거래·재회. 그녀 손등 힐링 문양이 흐려짐을 도현이 알아챔(죽음 미래 복선).
- **등장**: 도현·한소이·곽노인.
- **심는 떡밥**: **F-07(한소이 손등 힐링 문양 2차 노출·흐려짐)**. EP01 P31이 1차이므로 EP03은 **"EP01보다 한 단계 더 흐려짐"**으로 연출 — EP01 수준(1차 밝기)으로 절대 되돌리지 않는다(F-07 단조 감소).
- **반전 슬롯**: F-07 심기(하강 복선 강화) + 최봉규 팀 충돌 예고.
- **EP03 클리프행어**: 최봉규 팀이 같은 게이트에 배정됨 확인(충돌 예고).
